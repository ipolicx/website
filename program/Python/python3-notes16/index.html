Python札记16_操作文件（with、read、readline、readlines)

  • 打开文件用到open函数，标准的语法为： open(name[,mode[,buffering]])
  • open函数的文件名是必须的，模式和缓冲参数是可选的；例如，打开 c:\text的文件 a.txt： x = open(r'c:\text\a.txt')
  • 用读的模式打开文件，如果不存在就会报错。
    image.png

文件打开模式 r&#x3001;w&#x3001;a对应基本的 &#x53EA;&#x8BFB;、 &#x53EA;&#x5199;、 &#x8FFD;&#x52A0;&#x6A21;&#x5F0F;；
b&#x3001;t&#x3001;+&#x3001;U对应 &#x4E8C;&#x8FDB;&#x5236;&#x6A21;&#x5F0F;、 &#x6587;&#x672C;&#x6A21;&#x5F0F;、 &#x8BFB;&#x5199;&#x6A21;&#x5F0F;、 &#x901A;&#x7528;&#x6362;&#x884C;&#x7B26;。

模式 含义 是否报错 位置 r 只读 报错 文件开头 r+ 读+写 报错 文件开头 w 只写 创建 清空 文件开头 w+ 写+读 创建 清空 文件开头 a 追加写 创建 文件结尾 a+ 追加写+读 创建 文件结尾

  • f = open('a.txt', 'w') //用写的方式打开 a.txt文件，并且赋给f
  • f.write('hello,') // write写入括号中的内容
  • f.write('i play python') //写入内容，但是内容是追加在 hello之后的
  • f.close() //调用 close方法，关闭文件

四、文件读取操作方法

文件的读取主要是三种方式： read&#x3001;readline&#x3001;readlines，每次读取都有指针伴随着移动，用 seek()函数能够查看指针的偏移量，下面进行详细的讲解：

read

  • file.read(size)：参数 size可省略
  • 按照 &#x5B57;&#x8282;读取，并返回的是 &#x5B57;&#x7B26;&#x4E32;
  • 将文件内容从指针开始读取到文件结束
  • 全部读取，返回的是 &#x5B57;&#x7B26;&#x4E32;

readline

  • 从指针位置读取一行，直到最后一行
  • 到了最后一行，如果还读取则返回 &#x7A7A;&#x5B57;&#x7B26;&#x4E32;， &#x4E0D;&#x62A5;&#x9519;

readlines

  • 将文件按照行读取，放到一个列表中，用for进行遍历
  • 列表中的每个元素都是文件的一行，并且都是字符串
  • readlines利用 for循环读取和 for i in f直接获取的区别是：
    • 前者先执行f.readlines，将文件的每行先存入内存中；
    • 后者没有，一次性将文件全部读入内存中；
