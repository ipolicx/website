Python札记17_初识函数function

在数学中，一个函数就是一种 &#x6620;&#x5C04;关系。 在Python中一切皆是对象，函数也是对象，那么函数有自己的属性。在 Python中用句点（英文的句号.）来表示对象的属性， function函数的定义，通过一个例子理解如下：

初识函数

  • 在 Python中 &#x5192;&#x53F7;和 &#x7A7A;&#x683C;很重要
  • 第一行
    • def是函数的开始，告诉 Python解析器要声明一个函数
    • add_function：函数名字；不能是 Python中的关键字
    • add_function：是一个对象，function类；只有对象才可以与变量建立引用关系
    • x&#xFF0C;y：传入函数中的参数，要写在括号里面
    • 参数x，y没有特殊的类型；在Python中， 变量无类型，对象才有类型
    • 第一行整体被称为函数头
  • 第二行
    • 缩进4个空格， Python的规定，否则报错
    • 称为函数体，需要执行的内容
  • 第三行
    • 缩进4个空格
    • 返回有调用这个函数的地方；如果不写，默认是 return None
  • 第五行：固定写法
  • 第六行：
    • 传入两个值 2,3，调用之前的函数
    • 相当于是把 2,3分别放在 a,b的位置上
  • 第七行：打印输出 result值

命名

Python的命名规则为：

  • 变量全部小写，由下划线连接各个单词；
  • 变量名不能与Python的关键字冲突
  • 文件名全部小写，可使用下划线
  • 函数名：主要是驼峰体和匈牙利命名。Python中一般采用小写，可使用下划线

参数和变量

参数分为形参和实参。

  • 形参：
    • 函数名后面的括号里面如果有参数，称之为 &#x5F62;&#x53C2;
    • 形参表示一个值，过程希望调用它时候传递该值
    • 定义函数的时候，括号内部指定形参列表
    • 每个形参可以作为过程内部的变量

image.png

  • 实参：
    • 调用函数的时候，给函数提供过的参数称之为 &#x5B9E;&#x53C2;，或者称之为 &#x53C2;&#x6570;
    • 调用时候传递给形参的值
    • 实参没有表达式名字，每个实参都是一个表达式
    • 函数参数传递的对象引用（ &#x52A1;&#x5FC5;&#x7262;&#x8BB0;）

Python札记16_操作文件（with、read、readline、readlines)

  • 打开文件用到open函数，标准的语法为： open(name[,mode[,buffering]])
  • open函数的文件名是必须的，模式和缓冲参数是可选的；例如，打开 c:\text的文件 a.txt： x = open(r'c:\text\a.txt')
  • 用读的模式打开文件，如果不存在就会报错。
    image.png

文件打开模式 r&#x3001;w&#x3001;a对应基本的 &#x53EA;&#x8BFB;、 &#x53EA;&#x5199;、 &#x8FFD;&#x52A0;&#x6A21;&#x5F0F;；
b&#x3001;t&#x3001;+&#x3001;U对应 &#x4E8C;&#x8FDB;&#x5236;&#x6A21;&#x5F0F;、 &#x6587;&#x672C;&#x6A21;&#x5F0F;、 &#x8BFB;&#x5199;&#x6A21;&#x5F0F;、 &#x901A;&#x7528;&#x6362;&#x884C;&#x7B26;。

模式 含义 是否报错 位置 r 只读 报错 文件开头 r+ 读+写 报错 文件开头 w 只写 创建 清空 文件开头 w+ 写+读 创建 清空 文件开头 a 追加写 创建 文件结尾 a+ 追加写+读 创建 文件结尾

  • f = open('a.txt', 'w') //用写的方式打开 a.txt文件，并且赋给f
  • f.write('hello,') // write写入括号中的内容
  • f.write('i play python') //写入内容，但是内容是追加在 hello之后的
  • f.close() //调用 close方法，关闭文件

四、文件读取操作方法

文件的读取主要是三种方式： read&#x3001;readline&#x3001;readlines，每次读取都有指针伴随着移动，用 seek()函数能够查看指针的偏移量，下面进行详细的讲解：

read

  • file.read(size)：参数 size可省略
  • 按照 &#x5B57;&#x8282;读取，并返回的是 &#x5B57;&#x7B26;&#x4E32;
  • 将文件内容从指针开始读取到文件结束
  • 全部读取，返回的是 &#x5B57;&#x7B26;&#x4E32;

readline

  • 从指针位置读取一行，直到最后一行
  • 到了最后一行，如果还读取则返回 &#x7A7A;&#x5B57;&#x7B26;&#x4E32;， &#x4E0D;&#x62A5;&#x9519;

readlines

  • 将文件按照行读取，放到一个列表中，用for进行遍历
  • 列表中的每个元素都是文件的一行，并且都是字符串
  • readlines利用 for循环读取和 for i in f直接获取的区别是：
    • 前者先执行f.readlines，将文件的每行先存入内存中；
    • 后者没有，一次性将文件全部读入内存中；

Python札记15_break、continue、else

break：在某个地方中断循环，跳出循环体
continue：从当前位置（ continue所在位置）中跳到循环体 &#x6700;&#x540E;&#x4E00;&#x884C;&#x7684;&#x540E;&#x9762;，不执行最后一行。对于一个循环体来说， &#x6700;&#x540E;&#x4E00;&#x884C;&#x7684;&#x540E;&#x9762;&#x5C31;&#x662F;&#x5F00;&#x59CB;。通过例子来体会两种语句
else：主要适合if或者while等进行联合使用。

理解结果：

  • 最开始 a=9不满足 if，执行 else的 print语句，同时 a变成8进入第二次循环。

  • 第二次循环 a=8，满足 if， a执行自减1操作变成7，遇到 continue，跳到循环开始地方 while语句，带着 a=7进入第三次循环。

  • 第三次循环 a=7，不满足if，执行 else的 print语句，同时 a变成6进入第四次循环。

  • 第四次循环 a=6，满足 if， a执行自减1操作变成5，遇到 continue，跳到循环开始地方 while语句，带着 a=5进入第五次循环。
    ……

  • 最后一次， a=1不满足if，执行 else语句，同时 a自减变成0，不再满足 while语句条件，循环结束。

while…else…

按照步骤来理解：

  • a=0满足 while条件，执行第一个 print，并且自加1变成了1，进去第二次循环

  • a=1满足 while条件，执行第一个 print，并且自加1变成了2，进去第三次循环

  • a=2满足 while条件，执行第一个 print，并且自加1变成了3，进去第四次循环
    ……

  • a=4满足 while条件，执行第一个 print，并且自加1变成了5，进去第六次循环

  • a=5满足 while条件，执行 else语句，循环结束。
